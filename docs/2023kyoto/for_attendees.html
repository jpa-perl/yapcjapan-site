YAPC::Kyoto 2023 参加者の皆様へ

会場について

会場は京都リサーチパーク東地区1号館です。地図は公式サイトをご覧ください。

受付は東地区1号館アトリウムとなります。トークセッションは以下の部屋を利用します。

  • Scrapboxホール by Helpfeel (アトリウム)
  • Gyazoホール by Helpfeel (サイエンスホール)
  • 土俵 (G会議室)

なお、Scrapboxホール by Helpfeel (アトリウム)と、Gyazoホール by Helpfeel (サイエンスホール) / 土俵 (G会議室)の間は移動に5分程度必要です。登壇者の方は、発表開始の5分前にはトークセッション会場に到着できるようご注意ください。

新型コロナウイルス感染症(COVID-19)対策

2023年3月13日より、マスクの着用に関しては個人の判断に委ねるとの方針が政府より発表されております。 しかしながら、YAPC::Kyoto 2023は全国各地から300名近い参加者が集まるイベントであること、質疑応答や参加者同士のコミュニケーションなど、会話の機会が多いイベントであることを鑑み、参加者の皆様には以下の対応を推奨いたします:

  • マスク着用の推奨
  • ワクチン接種の推奨 (｢推奨｣のため、当日接種証明書など確認することはいたしません)
  • 会場での間隔をあけた着席など、ソーシャルディスタンス確保の推奨
  • 消毒液などを利用した手指の消毒の推奨

また、スタッフにより適宜以下の対策を行います:

  • 休憩時間を利用した換気の実施 (花粉症の季節ではありますが、何卒ご了承ください)
  • アルコールタオルなどを利用したテーブルなどの清掃

注意事項

  • 参加者の皆様には、Code of Conductの遵守をお願いいたします
  • 会場への危険物の持ち込みはご遠慮ください
  • 会場ではスタッフの案内/指示に従ってください
  • 参加に際しては必ず最初に受付を行い、受付にて配布するストラップを見える位置で着用してください。ストラップの着用がない場合、スタッフがお声がけする場合がございます
  • Code of Conductに反する行為や危険な行為を見かけられた、または被害に遭った際は、お近くのスタッフにお声がけください
  • 会場は飲食禁止です(ランチセッションを除く)
  • 東地区1号館の3階はオフィスルームのため、立入禁止です

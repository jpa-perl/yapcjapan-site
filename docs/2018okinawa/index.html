About

YAPC::Japanがついに沖縄に!

YAPCはYet Another Perl Conferenceの略で、一般社団法人 JPAが主催する、Perlを軸としたITに関わる全ての人のためのカンファレンスです。
Perlだけにとどまらない多様な分野のトークセッションを通じ、情報交換や交流を行っていただくことで、
JPAのミッションである「日本におけるPerlコミュニティを脈々と続くものとして」盛り上げていくことが目標です。
今回のYAPC::Okinawaは「架け橋」を意味する、「万国津梁」をテーマに掲げて開催します。

  • ・いままで使ってきた、そしてこれから使っていくPerlの話
  • ・「こんな場所で役に立つ!」Perlで取得した技術
  • ・Perlと出会った事で広がった技術やコミュニティの話

といった内容を皆様にお届けできるよう、様々な企画・トークを準備してお待ちしております！
皆さま、こぞってご参加ください！

☆ ゲストスピーカー・トークスケジュール等はWebページ・Facebookにてお知らせいたします。併せてご覧ください。
☆ 前夜祭、懇親会についても開催を予定しております。詳細が決まり次第、順次お知らせいたします。

Special Session

Perl's work inside the company (Perl のお仕事)

表立った話題にはなることは少ない (外部からは見えにくい) けれど、その役割をしっかりとこなしている企業の中での「Perlのお仕事」の「今」を広く共有していただくスペシャルセッションです。
往時に比べれば、あれもこれもといった処理をやたらと担うことは少なくなっているPerlですが、「それでも」とPerlが採用されている仕事があるのなら、そこにはきっとPerlならではの何かがあるに違いなし！
厳選された 4 社のセッション。お聞き逃しなく！

登壇者(登壇順、敬称略)

LINE株式会社

谷口 公一 @nipotan

LINE が提供する LINE NEWS というニュースサービスの裏側は、実は Perl で開発された Web アプリケーションが中枢を担っています。Perl で実装された Web アプリケーションとしては、利用者数やトラフィックという点において、国内有数の規模になってきました。

「企業内での Perl」の一例として「LINE NEWS を支える Perl」というテーマで

  • LINE NEWS 開始から今日までの沿革と、システムという側面から見た沿革
  • livedoor 時代から培われた Perl 開発の経験と実績
  • ORM の今と昔
  • サービス特性に起因する問題点と高可用性の実現
  • 高速を求めた一つの形
  • Perl とそれ以外のシステムの共存
  • LINE NEWS のこれから
について、事例を交えてご紹介します。

株式会社メルカリ

長野 雅広 @kazeburo

メルカリではサーバ側の開発言語としてPHP、Go言語を、また機械学習ではPythonなどを利用しています。Perlはというと残念ながら表立ってサービスを構築する言語として活躍する機会はありません。しかしながらPerlで作られた小さなスクリプトがサービスの安定稼働を支え、また裏方として大規模なトラフィック・データを処理するミドルウェアとしても利用されています。

このトークでは、そんな大小のPerlの事例や、なぜPerlなのか、今後どうなっていくのかについて紹介したいと思います。

S2ファクトリー株式会社

吉津 卓保 @n0442

S2 ファクトリーは受託をメインとしたシステム開発会社です。これまで 10 年以上に渡り様々なシステムやサービスのバックエンドを開発してきました。

インスタレーションなどメディアアートのシステム、ハイトラフィックな WEB キャンペーンの応募システム、カスタムユニフォームのシミュレーションサイト、オンラインと店舗のオムニチャンネルの EC サービス、などなどその事例は多岐に渡ります。

いずれもクライアントの要望を元にスクラッチで作られたシステムであり、そのほとんどが Perl で書かれています。いくつかの事例をピックアップして、Perl が実際にどういった形で活躍しているのか、なぜ Perl を採用し続けているのか、今後も Perl は活躍していけるのか、といったことをお話ししたいと思います。

株式会社フリークアウト

本間 雅洋 @hiratara

国内初のDSPであるフリークアウトのシステムは大部分がPerlで書かれています。
DSPの心臓部とも言えるbidderもPerlで書かれており、現在も弊社ビジネスの中心をなすシステムとして稼働しています。

Perlで書かれた弊社の様々なサービスを通して、Perlを使った開発の実用性に触れると共に、プロダクトを取り巻くIT環境のこの7年での変化や、弊社プロダクトの開発に携わるエンジニア像、そして、Perl以外でどのような言語やミドルウェアの力を借りているか、など、エンジニアの立場から現在のPerlを使った開発の生の現場を、20分間で話せる範囲で紹介します。
